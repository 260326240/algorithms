# JSP相关知识点





1. jsp是什么  
**java server page 嵌入java的html页面，也就是servlet**
	

2. servlet是什么 
 **小服务程序或服务连接器，用Java编写的服务器端程序，主要功能在于交互式地浏览和修改数据，生成动态Web内容。** 
3. jsp和servlet的区别  
	### jsp  
		(1)优点：页面表现方便
	   	(2)缺点：逻辑处理麻烦     
	### servlet
		(1)优点：逻辑处理方便
		(2)缺点：页面表现麻烦
#### 所以一般项目都采用  
### **jsp+servlet+javabean** ###
